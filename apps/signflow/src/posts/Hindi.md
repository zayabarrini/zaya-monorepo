# Grammatical Rules

Aqui estão **30 regras gramaticais essenciais do Hindi**, cobrindo aspectos como **ordem das palavras, gênero, verbos, casos, pós-posições e peculiaridades do idioma**:

---

### **1. Ordem Básica das Palavras (SOV)**

- O Hindi segue a estrutura **Sujeito-Objeto-Verbo (SOV)**:
  - _मैं सेब खाता हूँ।_ (Main seb khātā hū̃) → "Eu como uma maçã."

### **2. Gênero Gramatical (Masculino vs. Feminino)**

- **Masculino**: Terminações com **-आ (ā)** (ex: _लड़का_ / laṛkā / "menino").
- **Feminino**: Terminações com **-ई (ī)** (ex: _लड़की_ / laṛkī / "menina").
- Exceções: _माता_ (mātā / "mãe") é feminino, mas termina em **-आ**.

### **3. Concordância de Gênero e Número**

- Adjetivos e verbos concordam com o gênero/número do sujeito:
  - _लड़का खाता है।_ (Laṛkā khātā hai) → "O menino come."
  - _लड़की खाती है।_ (Laṛkī khātī hai) → "A menina come."

### **4. Pluralização**

- Masculino: **-आ → -ए** (ex: _लड़के_ / laṛke).
- Feminino: **-ई → -इयाँ** (ex: _लड़कियाँ_ / laṛkiyā̃).

### **5. Casos Gramaticais (Pós-Posições)**

- O Hindi usa **pós-posições** (como preposições, mas após o nome):
  - **को (ko)**: Acusativo/Dativo (objeto direto/indireto).
    - _मैंने उसे देखा।_ (Maine use dekhā) → "Eu vi **ele**."
  - **से (se)**: Instrumental/Ablativo ("com"/"de").
    - _मैंने कलम से लिखा।_ (Maine kalam se likhā) → "Escrevi **com** uma caneta."

### **6. Verbos e Tempos**

- **Presente**: Raiz + **-ता/-ती** (ex: _खाता/खाती_).
- **Passado**: Raiz + **-या/-ई** (ex: _खाया/खाई_).
- **Futuro**: Raiz + **-गा/-गी** (ex: _खाऊँगा/खाऊँगी_).

### **7. Auxiliares de Respeito (Honoríficos)**

- **है (hai)** → Informal.
- **हैं (hain)** → Formal/respeitoso (ex: _वह आते हैं।_ / "Ele vem [respeitoso]").

### **8. Pronomes Pessoais**

- **मैं (main)** → "Eu" (informal).
- **हम (ham)** → "Nós" ou "Eu" (formal).

### **9. Pronomes Demonstrativos**

- **यह (yah)** → "Este/isso" (perto).
- **वह (vah)** → "Aquele/aquilo" (longe).

### **10. Negação**

- **नहीं (nahī̃)** antes do verbo:
  - _मैं नहीं जाता।_ (Main nahī̃ jātā) → "Eu não vou."

### **11. Perguntas**

- **क्या (kyā)** no início para perguntas sim/não:
  - _क्या तुम हिंदी बोलते हो?_ (Kyā tum Hindi bolte ho?) → "Você fala Hindi?"

### **12. Advérbios de Tempo**

- **आज (āj)** → "Hoje".
- **कल (kal)** → "Amanhã"/"Ontem" (depende do contexto).

### **13. Conjunções**

- **और (aur)** → "E".
- **लेकिन (lekin)** → "Mas".

### **14. Partículas de Ênfase**

- **ही (hī)** → "Apenas/só":
  - _मैं ही जाऊँगा।_ (Main hī jāū̃gā) → "Só **eu** irei."

### **15. Verbos Compostos**

- **Verbo principal + auxiliar**:
  - _लिख देना_ (likh denā) → "Escrever (completamente)".

### **16. Voz Passiva**

- **Verbo + जाना (jānā)**:
  - _किताब पढ़ी जाती है।_ (Kitāb paṛhī jātī hai) → "O livro é lido."

### **17. Causativos**

- **-वाना (-vānā)**:
  - _पिलाना_ (pilānā) → "Fazer beber".

### **18. Imperativo**

- **Raiz + -ओ/-इये**:
  - _जाओ!_ (jāo!) → "Vá!" (informal).
  - _जाइये!_ (jāiye!) → "Vá!" (formal).

### **19. Condicionais**

- **अगर (agar)...तो (to)**:
  - _अगर बारिश होगी, तो मैं नहीं आऊँगा।_ (Agar bārish hogī, to main nahī̃ āū̃gā) → "Se chover, não irei."

### **20. Pronomes Possessivos**

- **मेरा (merā)** → "Meu" (masculino).
- **मेरी (merī)** → "Minha" (feminino).

### **21. Numerais**

- **एक (ek)** → "1".
- **दस (das)** → "10".

### **22. Expressões de Tempo**

- **सुबह (subah)** → "Manhã".
- **रात (rāt)** → "Noite".

### **23. Advérbios de Lugar**

- **यहाँ (yahā̃)** → "Aqui".
- **वहाँ (vahā̃)** → "Lá".

### **24. Comparativos**

- **से (se)**:
  - _राम सीता से लम्बा है।_ (Rām Sītā se lambā hai) → "Ram é mais alto que Sita."

### **25. Superlativos**

- **सबसे (sabse)**:
  - _यह सबसे अच्छा है।_ (Yah sabse acchā hai) → "Isso é o melhor."

### **26. Partículas de Cortesia**

- **जी (jī)** → "Sim/senhor":
  - _हाँ जी!_ (Hā̃ jī!) → "Sim, senhor!"

### **27. Onomatopeias**

- **खटाखट (khaṭākhaṭ)** → "Barulho de batida".

### **28. Empréstimos do Sânscrito/Inglês**

- **स्कूल (skūl)** → "Escola".
- **टेलीविजन (ṭelīvijan)** → "Televisão".

### **29. Contrações Informais**

- **हो (ho)** → "Está bem?" (coloquial).

### **30. Regras de Pronúncia**

- **अनुस्वार (ṁ)** e **चंद्रबिंदु (ñ)** nasalizam vogais:
  - _हँसना (hãsnā)_ → "Rir".

---

### **Dica Final**:

- O Hindi **não usa artigos** ("o", "a"). O contexto define o significado.
