# tone shifts

### 高级变调规则与例子

#### 一、三声变调（双三声）

1. 单字 → 词语变化：

   - 我 (wǒ) → 我们 (wómen)
   - 你 (nǐ) → 你好 (níhǎo)
   - 好 (hǎo) → 好吃 (háochī)

2. 三字组合：
   - 展览馆 → zhánlǎnguǎn
   - 洗脸水 → xíliǎnshuǐ

#### 二、"一"的复杂变调

1. 在第四声前变第二声：

   - 一个 (yígè)
   - 一定 (yídìng)

2. 在第一/二/三声前变第四声：

   - 一些 (yìxiē)
   - 一年 (yìnián)
   - 一起 (yìqǐ)

3. 在动词中间变轻声：
   - 看一看 (kànyikàn)
   - 想一想 (xiǎngyixiǎng)

#### 三、"不"的变调

1. 在第四声前变第二声：

   - 不对 (búduì)
   - 不要 (búyào)

2. 其他情况不变：
   - 不能 (bùnéng)
   - 不好 (bùhǎo)

1. AA 式：

   - 慢慢 (mànmān)
   - 甜甜 (tiántiān)

2. AABB 式：
   - 高高兴兴 (gāogao-xīngxing)
   - 清清楚楚 (qīngqing-chǔchu)

#### 五、特殊口语变调

1. 连读变化：

   - 豆腐 → dòufu (腐变轻声)
   - 耳朵 → ěrduo (朵变轻声)

2. 习惯变调：
   - 两个 (liǎng → liǎ)
   - 三个 (sān → sá)

#### 六、对比练习

1. 原调 vs 变调：

   - 你好 (nǐhǎo → níhǎo)
   - 雨伞 (yǔsǎn → yúsǎn)

   - 不是 (bùshì → búshì)

# Pinyin vs. English Sounds

### 拼音与英语发音对比

#### 相似但不同的音

1. **b/p/m/f**

   - 英语更重爆破（如"**b**ad"），拼音更轻（**b**àba 爸爸）
   - 对比：英语"**p**en" vs 拼音"**p**ào 跑"

2. **j/q/x**

   - 英语无对应音：
     - **j**ī 鸡 → 类似"**gee**"但舌面贴硬腭
     - **q**ì 气 → 类似"**chee**"但送气更强
     - **x**ī 西 → 类似"**she**"但舌尖下垂

3. **zh/ch/sh/r**
   - 英语"**j**ump" vs 拼音"**zh**ōng 中"（卷舌）
   - 英语"**ch**urch" vs 拼音"**ch**ē 车"（舌头更后缩）

#### 易混淆音对比

| 拼音  | 英语近似                  | 例子        | 常见错误         |
| ----- | ------------------------- | ----------- | ---------------- |
| **c** | **ts** in "cats"          | **c**ài 菜  | 勿发成"**th**ai" |
| **z** | **dz** in "kids"          | **z**ì 字   | 勿发成"**z**oo"  |
| **x** | **sh** + 咧嘴             | **x**iǎo 小 | 勿发成"**k**s"   |

# Tone Change Rules

### 汉语变调规则大全

#### 一、三声变调（最核心规则）

1. **三声 + 三声 → 二声 + 三声**

   - 你好 (ní hǎo) ← 原调：nǐ hǎo
   - 水果 (shuí guǒ) ← 原调：shuǐ guǒ

2. **三声 + 非三声 → 半三声（低平调）**
   - 北京 (běi jīng) ← 原调：běi jīng
   - 老师 (lǎo shī) ← 原调：lǎo shī

#### 二、"一"的变调

| 位置           | 变调 | 例子                | 原调 |
| -------------- | ---- | ------------------- | ---- |
| 第四声前       | 二声 | 一个 (yí gè)        | yī   |
| 第一/二/三声前 | 四声 | 一天 (yì tiān)      | yī   |
| 动词中间       | 轻声 | 看一看 (kàn yi kàn) | yī   |

#### 三、"不"的变调

- **第四声前变二声**：  
  不对 (bú duì) | 不要 (bú yào)
- **其他情况不变**：  
  不能 (bù néng) | 不好 (bù hǎo)

#### 四、重叠词变调

1. **AA 式形容词**：

   - 慢慢 (màn m**a**n) ← 第二字轻声
   - 高高 (gāo g**a**o)

2. **AABB 式**：
   - 高高兴兴 (gāo gao - xīng xing)
   - 干干净净 (gān gan - jìng jing)

#### 五、特殊变调

1. **口语习惯**：

   - 两个 (liǎng → liǎ)
   - 三个 (sān → sá)

2. **否定连读**：
   - 是不是 (shì bu shì)
   - 去不去 (qù bu qù)

#### 六、对比练习

1. 原调 vs 变调：

   - 雨伞 (yǔ sǎn → yú sǎn)
   - 早起 (zǎo qǐ → záo qǐ)

2. 否定对比：
   - 不是 (bù shì → bú shì)
   - 不要 (bù yào → bú yào)

#### 七、常见例外

- 固定词组不变调：  
  委员会 (wěi yuán huì，三声不変)
- 姓氏不变调：  
  李伟 (Lǐ Wěi，保持原调)
