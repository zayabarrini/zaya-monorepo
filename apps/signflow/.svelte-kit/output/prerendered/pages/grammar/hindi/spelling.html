उच्चारण 🗣️

VowelPronunciation
अ (a)like 'u' in "but"
आ (ā)like 'a' in "father"
इ (i)like 'i' in "sit"
ई (ī)like 'ee' in "see"
उ (u)like 'u' in "put"
ऊ (ū)like 'oo' in "moon"

संस्कृत 🌸

Devanagari is an abugida where each consonant inherently contains the vowel अ (a). Diacritics modify this inherent vowel.

क + ा = का क + ि = कि क + ु = कु

विशेष ⭐

अनुस्वार (ं): कं (kaṁ) - nasalization

विसर्ग (ः): कः (kaḥ) - voiceless breath

चन्द्रबिन्दु (ँ): काँ (kā̃) - nasalized vowel

🔧 संयुक्ताक्षर निर्माण (Formation Rules)

हलंत का प्रयोग (Using Halant)

क् + ष = क्ष क् (halant removes 'a') + ष = क्ष
त् + र = त्र त् + र = त्र (tra)

र के विशेष रूप (Special Ra Forms)

र् + क = र्क Ra as first element
क् + र = क्र Ra as last element

रेफ (Reph)

र् + क = र्क Ra above the consonant
र् + य = र्य Ra above ya

महत्वपूर्ण नियम (Important Rules)

  • चंद्रबिंदु (ँ) - nasalization: पाँच (pāñch)
  • अनुस्वार (ं) - nasal sound: पंद्रह (pandrah)
  • विसर्ग (ः) - voiceless breath
  • हलंत (्) - suppresses inherent vowel

विशेष उच्चारण (Special Pronunciation)

  • ड़ - retroflex flapped: सड़सठ (saṛsaṭh)
  • ढ़ - aspirated retroflex flapped
  • क्ष - क + ष: इक्यासी (ikyāsī)
  • ज्ञ - ज + ञ
