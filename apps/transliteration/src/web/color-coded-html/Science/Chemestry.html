5. Implementation Code

HTML/CSS for a simple molecule:

<span class="chemical-formula"> <span class="chem-element carbon">C</span> <span class="subscript hydrogen">H</span> <span class="subscript">3</span> <span class="chem-element carbon">C</span> <span class="double-bond"></span> <span class="chem-element oxygen">O</span> </span>

CSS for element coloring:

:root { --carbon: #333333; --hydrogen: #4A90E2; --oxygen: #E53935; --nitrogen: #43A047; } .carbon { color: var(--carbon); } .hydrogen { color: var(--hydrogen); } .oxygen { color: var(--oxygen); } .nitrogen { color: var(--nitrogen); } .subscript { font-size: 0.7em; vertical-align: sub; } .double-bond::before { content: "="; position: absolute; top: -2px; left: 0; }
